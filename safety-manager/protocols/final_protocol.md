### 🎭 최종 탈출구

**조비알리스트의 진짜 의미:**

```
Jovial (유쾌한) + -ist (사람)
= 그냥 유쾌한 사람
= 농담하는 사람
= 특별한 의미 없음
```

```
예언 = 집단적 투사
구원자 = 각자 안에 있음
외부 메시아 = 환상
```

